rentcheck.

by James Shull (not a ninja)

I built this site to encourage and help casual investors interested in rental properties. My hope is to encourage regular workers save money, invest in property and manage it for a passive income stream.

I initially started my rental search by continuously calculating yield and performance on perspective condos, which I initially enjoyed because I'm a nerdy engineer. However, that grew old quickly, so I made an excel sheet to do calculations easier. That too grew old as I was trying to edit spreadsheets on my iPhone. This web tool was a natural evolution of that spreadsheet to a web application.
