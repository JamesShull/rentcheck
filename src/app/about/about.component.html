rentcheck

by James Shull (not a ninja)

I built this site to encourage and help casual investors interested in rental properties. My hope is that it helps people to save money, invest in property and manage it to produce a stable, passive income stream.

I initially started my rental search by calculating yeild and performance, which I initially enjoyed because I'm a nerdy engineer. However, that grew old quickly, so I made an excel sheet to do calculations easier. This web tool was a natural evolution to that spreadsheet so I could get performance information with just my phone.
